# 📚 StudyFlow Mobile

StudyFlow Mobile is a professional, offline-first productivity application designed to help students manage their schedules, track assignments, and stay organized. 

Built with modern mobile technologies, StudyFlow ensures your academic life remains accessible anytime, anywhere, even without an internet connection.

---

## ✨ Features

* **📴 Offline First:** Access your schedules and tasks seamlessly without an internet connection using local storage.
* **📅 Weekly & Monthly Planner:** Easily track your weekly recurring classes, assignments, and upcoming exams.
* **🔔 Smart Notifications:** Stay updated with timely alerts and reminders for your critical tasks.
* **🎨 Beautiful UI/UX:** Built with React Native & Expo, offering fluid animations and a distraction-free, intuitive interface.

## 🛠️ Technologies Used

* **Framework:** [React Native](https://reactnative.dev/) & [Expo](https://expo.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Local Data:** `@react-native-async-storage/async-storage`
* **Build System:** EAS (Expo Application Services)
* **Routing:** Expo File-Based Routing
