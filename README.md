# StudyFlow Mobile

StudyFlow Mobile is a professional, offline-first, productivity application designed to help students manage their schedules, track assignments, and stay organized. 

## Features
- **Offline First**: Access your schedules and tasks anytime, anywhere without an internet connection.
- **Weekly & Monthly Planner**: Seamlessly track your weekly recurring classes, assignments, and exams.
- **Smart Notifications**: Stay updated with timely alerts for your upcoming tasks.
- **Beautiful UI/UX**: Built with React Native & Expo, offering fluid animations and intuitive interfaces.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- npm or yarn
- [Expo CLI](https://expo.dev/)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npx expo start
   ```

## Development

You can start developing by editing the files inside the `app` directory. This project uses Expo file-based routing.
