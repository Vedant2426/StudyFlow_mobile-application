# StudyFlow Mobile 📚

**StudyFlow Mobile** is a modern, offline-first student productivity app built with **React Native + Expo**.
It helps students manage classes, assignments, deadlines, and study sessions in one clean mobile experience.

Designed for speed, simplicity, and real academic use.

---

## ✨ Key Features

### 📅 Planner & Timetable

* Weekly class schedule
* Monthly planning view
* Upcoming events tracking
* Organized academic workflow

### ✅ Assignment Manager

* Add, edit, delete tasks
* Due date tracking
* Mark tasks as completed
* Stay ahead of deadlines

### ⏱️ Study Timer

* Focus timer for study sessions
* Productivity tracking
* Better concentration habits

### 🔔 Smart Notifications

* Assignment reminders
* Upcoming class alerts
* Daily planning nudges

### 🌙 Theme Support

* Light mode
* Dark mode
* Clean modern UI

### 📱 Offline First

No internet required.
Your data stays available anytime, anywhere.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm or yarn
* Expo CLI

### Installation

```bash
git clone YOUR_REPO_LINK
cd studyflow-mobile
npm install
npx expo start
```
